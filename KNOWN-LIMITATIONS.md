# KNOWN LIMITATIONS — read before relying on this

**Status: REFERENCE / EXPERIMENTAL.** This repository contains an **open-source-intelligence snapshot and analysis** for a non-EU track (Cameroon). It is research material, not a deployed service.

This file is published deliberately. Doctrine: **claim ≤ proof** — we document where this is
incomplete rather than overclaiming. The commons underwent internal adversarial review rounds
plus an external review (RSpace); findings and fixes are public in the git history.

## Known open weaknesses
- **Snapshot, not monitoring.** Figures and rankings reflect the moment of research and are not
  refreshed. Re-check every cited indicator against its primary source before use.
- **Analysis, not endorsement.** Identifying a gap or an opportunity is not a claim that any
  engagement exists, is planned, or is appropriate.
- **No deployment.** Nothing here runs as a service; the surfaces are static documents.
- **Language coverage is partial** (PL/EN only) and terminology may not match local usage.

## What IS solid
- Every substantive statement traces to a cited source; where a source is missing it is marked
  as a gap rather than filled in. Apache-2.0; SBOM with per-file SHA-256.

## How to help
Try to break it and open an issue or PR with a reproducing input. That is exactly how the
limitations above were surfaced.
